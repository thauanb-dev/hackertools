# 🧰 Hackertools

Uma coleção de ferramentas web para análise de rede e informações técnicas, com interface moderna estilo cyber.

---

## 🚀 Funcionalidades

- 🔎 DNS Lookup  
  Consulta registros DNS (A, etc) de qualquer domínio

- 🌍 IP Tracker
  Identifica informações de IP (localização, país, etc)

- Crypto
  Ferramentas de Cripitografica
  - Base64
  - Hash SHA-1, SHA-256, SHA-512
  - JWT Decoder

---

## 🖥️ Preview

Interface inspirada em ferramentas de análise e pentest, com foco em simplicidade e usabilidade.
